Welcome to WuWei

WuWei is a software package for Common Lisp that makes building Ajaxified web sites in Lisp as close to effortless as possible.

"Wu wei" means "effortless doing" or "action without action".

The source is hosted at GitHub.

Features

  • Continuation-based AJAX user interfaces
  • Server-side high-level DOM operations (add/remove elements, visual fades, drag and drop)
  • High-level interfaces to in-place editing and autocomplete widgets
  • Login and session management
  • OAuth2 support
  • Runs in multiple Common Lisp implementations
  • Freely available under the MIT Open Source License

Examples and demos

  • Basic Ajax update machinery
  • Asynchronus results and error handling
  • Ajax forms and third party javascript libraries
  • Autocomplete field
  • Session variables and state maintenance
  • Answer to Paul Graham's ARC challenge
